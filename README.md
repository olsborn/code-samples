# Next.js App

This is a **Next.js** project bootstrapped with `create-next-app`.

---

## Getting Started

### Requirements
- **Node.js** (recommended LTS)
- **npm**

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```
Open `http://localhost:3000` in your browser to see the result. While in development mode, edit `app/page.js` (or `app/page.tsx`) and the page will auto-update as you save.

---

## Build and Run (production)

### Build (TurboPack-optimized)
```bash
npm run build
```

### Start the compiled app
```bash
npm start
```
Open `http://localhost:3000` in your browser to see the production build.

---

## Project structure (high level)
- **app/** — application routes and project pages 
- **components/** or **content/** — components container 
- **public/** — static assets  
- **package.json** — scripts and dependencies
